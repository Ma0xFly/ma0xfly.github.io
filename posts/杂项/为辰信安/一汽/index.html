tcpdump

  1. 登入车载网关/域控制器（带 root 权限）。
  2. 用 ip link 或 ifconfig 找到对应接口名，举例：
    • 以太网主干：eth0
    • 100BASE-T1 诊断口：enp3s0
    • 车载 VLAN 20：eth0.20
  3. 全量抓包（调试用，文件大）
tcpdump -i eth0 -s 0 -w /data/vehicle_full.pcap
  • -s 0 保留完整帧，防止截断。
  • 按 Ctrl+C 停止，生成 pcap 文件。

端口扫描

sudo nmap -sS -Pn -p- --min-rate 1000 192.168.97.143

解释：

  • -sS：半开 SYN 扫，最轻量。
  • -Pn：跳过主机发现，直接扫。
  • -p-：0-65535 全端口。
  • --min-rate 1000：每秒至少 1000 包，几分钟跑完。

扫完结果会列出开放端口、服务名。

sudo hping3 -A --flood -p 80 192.168.240.10 tcp-ack

sudo hping3 -S --flood -p 80 192.168.240.10

sudo hping3 -1 --flood 192.168.240.10
