无线电波

电磁波谱

性质

L=ft

**振幅：**指的是波的波动幅度 往往能量越大振幅越高 表现出来的信号强度就越高

**波长：**是波在一个振动周期传播的距离

（波长变在几公里到几米的范围 频率随之变成3000Hz 到100MHz 就变成了FM广播和AM广播的无线电波 半米左右到10厘米左右的波长 就是我们现在的4G风格网络的无线电波 变成波长12厘米 对应的是2.4GHz的电磁波 和6厘米对应的是5GHz的电磁波 2.4和5GHz的电磁波 就是我们常用的WiFi的无线电波）

**频率:**指的是波的周期 一秒钟通过一个完整的波长 就是一个Hz 1000赫就是一秒钟通过1000个波长 1兆赫就是100万赫兹 1GHz就是10亿赫兹

  • 1 Hz = 1 次/秒
  • 1 MHz = 1,000,000 Hz（百万赫兹）
  • 1 GHz = 1,000,000,000 Hz（十亿赫兹）

**速度：**电磁波在真空中以光速传播 在空气界之中会稍微慢一点点 但是依然很接近光速

（超低频 如说3赫兹的电磁波的波长能长达10万公里 虽然说尺度很大 但是请电磁波是以光速传播的 即便是10万公里的波长 一秒钟传播的距离仍然能达到30万公里 ，即10万公里长的波长的电磁波一秒钟也能传播三个波播的距离）

频段（Band）

定义： 频段就是一段特定范围的频率，像“2.4GHz~2.5GHz”这样的区间，叫做一个“频段”。

常见的无线频段：

名称 频段范围 应用举例
2.4GHz 2.400–2.4835GHz WiFi、蓝牙、微波炉
5GHz 5.150–5.825GHz WiFi（高速）
868/915MHz ~868MHz/915MHz LoRa、Zigbee等低功耗设备
24GHz、60GHz 更高频段 雷达、高速WiFi等

注意：不同频段有不同的穿透能力、距离、速度。2.4GHz 穿墙强但速度慢，5GHz 速度快但穿墙差。

蓝牙工作的波长范围为［120.7mm - 124.9mm］。波长（λ）和频率（f）之间的转换可以使用公式：波长 = 光速（c）/ 频率。即：λ = c / f 其中，光速 c 约为 299，792，458 米每秒（m/s）。 所以蓝牙的工作最高频率为： f = c / λ = 299，792，458 m/s / （120.7 x 10^-3 m） ≈2.487GHz 最低工作频率为： f = c / λ = 299，792，458 m/s / （124.9 x 10^-3 m） ≈2.40GHz 所以，蓝牙的工作频段为［2.4GHz ~ 2.48GHz］。

频率（Frequency）

定义：

指的是波的周期 一秒钟通过一个完整的波长 就是一个Hz 1000赫就是一秒钟通过1000个波长 1兆赫就是100万赫兹 1GHz就是10亿赫兹

频率是指一个信号每秒震动（振荡）的次数。单位是赫兹（Hz）。

  • 1 Hz = 1 次/秒
  • 1 MHz = 1,000,000 Hz（百万赫兹）
  • 1 GHz = 1,000,000,000 Hz（十亿赫兹）

假如你甩绳子甩得越快，那绳子振动得越密，就是频率越高。无线电信号也是一样——每秒振动得越快，频率越高。

带宽（Band width）

定义：

带宽是指某个频段内可以使用的“频率宽度”，也可以理解为一条路有多宽。一个频段中真正用于通信的一小段频率

应用举例（WiFi 的带宽）：

  • 2.4GHz WiFi 每个信道带宽通常是 20MHz
  • 5GHz WiFi 可以用 20、40、80、160MHz
    • 带宽越大，一次传输的数据越多，速度越快
    • 但带宽大了，越容易干扰、覆盖距离也可能变短

信道 （Channel）

定义：

信道是从一个频段中划分出的一小段带宽，用来进行通信。 它是“通信的专属小路”或“小车道”。

类比理解：

  • 频段 = 整条高速公路（比如 2.4GHz ~ 2.5GHz）
  • 带宽 = 每条车道的宽度（比如 20MHz）
  • 信道 = 在公路上划出来的一条车道（比如 2.412GHz ~ 2.432GHz）

举个 WiFi 的例子

WiFi 2.4GHz 频段信道划分：

  • 整个频段是：2.400 GHz ~ 2.4835 GHz
  • 每个信道占：20 MHz
  • 中心频率从 信道1（2.412GHz） 一直到 信道13（2.472GHz）
信道编号 中心频率
1 2.412 GHz
2 2.417 GHz
... ...
6 2.437 GHz
11 2.462 GHz
13 2.472 GHz

💡问题：这么多信道会重叠！怎么办？ 所以实际使用中，一般推荐选 1、6、11 这三个“非重叠信道”。

蓝牙的信道（也在2.4GHz）

蓝牙经典/低功耗（BLE）也用 2.4GHz，但和 WiFi 不一样：

  • 它划分了 79 个信道（每个1MHz），范围在 2.402 ~ 2.480 GHz
  • BLE（低功耗蓝牙）使用 40 个信道，每个宽 2 MHz
  • 蓝牙通信时会“跳频”：每隔一段时间换一个信道，避免干扰！

为什么信道重要？

信道让多个设备可以在同一个频段下：

  • 同时通信（像多个车道并行，而不是挤在一条道上相互干扰）
  • 避免冲突（互不干扰）
  • 提高频谱利用率（不浪费频率资源）
