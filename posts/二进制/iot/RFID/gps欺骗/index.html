GPS的工作原理

卫星通过他自己的星历，得知它当前的位置之后，计算出用户当前的位置的范围。但只通过一颗卫星去定位肯定是不准确的，同时需要四颗卫星才可以得到相对精准的一个位置。

Gps其实不仅仅可以定位，还可以授时。就是因为它里边有这个原子钟，它可以为地面的授时装置提供更加准确的时间。

GPS欺骗原理

GPS欺骗实际上就是通过无线电的设备，模拟一个卫星向用户的设备发送用户的位置信息，终端设备为了得到一个更准确的一个数据，那么会采用四颗以上信号比较强的卫星。设想一下，卫星距离我们的距离是非常远的，但是攻击者的发射装置离用户是特别近的。终端设备自然会选择利用攻击者的信号进行一个定位的请求。在经历这个过程之后，终端机接收到的那个定位便是攻击者想让他去接收的。

这个授时的欺骗是怎么回事呢？是因为gps在向用户发送坐标位置的时候，往往都会带有一个时间的信息。但是具体能不能用得上看情况。在早期的时候，我们的智能手机只要打开了位置服务就会自动获取时间，它会利用gps去授时。因为当时可能那个网络会存在很大的波动，可能没有现在这么快，所以说他的授予的时间可能会有误差，都会通过这样卫星的方式来获取到更加精准的时间。

初版HackRF 插件

一个完整的基于 Python 的 HackRF GPS 欺骗插件开发指南，基于 gps-sdr-sim 工具和 hackrf_transfer 命令。你将获得一个可以从命令行使用的 Python 工具，支持如下功能：

  • 自动生成 GPS 欺骗信号的 I/Q 数据（.bin 文件）；
  • 控制 HackRF 发射这些信号；
  • 可配置位置坐标、功率、频率等参数。

插件结构

gps_spoofer/
├── gps_sdr_sim/            # 克隆的 gps-sdr-sim 源码目录
│   └── gps-sdr-sim         # 编译后的可执行文件
├── gps_spoofer.py          # 主控制脚本
└── brdc0010.22n            # 星历文件（用于模拟）

运行实例

./gps_spoofer.py \
  --gpssim-path ./gps-sdr-sim \
  -e brdc0010.22n \
  -l 30.2865,120.0326,100 \
  -o gpssim.bin \
  -f 1575420000 \
  -s 2600000 \
  -x 20

json注释

    "input": [
        {
			"gpssim_path": "./gps-sdr-sim", // GPS 模拟工具的路径
            "ephemeris": "brdc1350.25n.gz",   // 星历文件路径
            "location": "29.55666693527179, 122.43254400514557, 10", // 纬度、经度,高度
            "output": "gpsspoof.bin",      // 输出的 GPS 欺骗信号文件
            "freq": 1575420000,            // 信号频率
            "samplerate": 2600000,         // 采样率
            "txgain": 20                   // 发射增益
        }
    ],

终版HackRF插件

https://chatgpt.com/share/68370ee1-a0b8-8002-8dfa-1893d8a61746

单独用gdr sim生成一个gps数据包，插件功能改为单独发送这个gps数据包

文件结构

gps_spoofer/
├── main.py                            # 主脚本（上面写的脚本）
├── test_case.json                     # 测试配置文件
├── utils/
│   └── Logger.py                      # 日志工具类
├── Service/
│   └── HTTPConnectionService.py       # 用于通知后端结果
├── Bean/
│   ├── PluginResult.py                # 定义 PluginResult 类型和状态常量

├── output/                            # 输出目录，生成的 gpsspoof.bin 可放这里
└── gpssim.bin                         # 你用 gps-sdr-sim 工具生成的 GPS 信号二进制文件
