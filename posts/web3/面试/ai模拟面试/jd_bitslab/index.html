Solidity 智能合约安全审计实习生 JD

来源：jd_bitslab.jpg

基本信息

  • 岗位：Solidity 智能合约安全审计实习生
  • 日薪：300-500 元/天
  • 地点：成都 · 武侯区 · 中和
  • 出勤：4 天/周
  • 实习周期：6 个月
  • 学历：本科
  • 图片中公司显示：智谷星图
  • 图片中招聘者显示：李先生

职位标签

  • 安全运维和保障
  • 渗透测试经验
  • 安全体系架构和研发
  • 攻防对抗经验
  • IT 审计经验

岗位职责

1. 智能合约代码审计

协助分析 Solidity 智能合约代码，识别潜在的安全漏洞，如重入攻击、溢出问题、权限控制缺陷等。

2. 漏洞报告撰写

整理审计发现，撰写清晰、详尽的安全审计报告，提出修复建议。

3. 安全工具使用

使用智能合约审计工具，如 Mythril、Slither、Echidna 等，进行静态和动态分析。

4. 技术研究

跟踪区块链和智能合约安全领域的最新技术、漏洞和攻击手法，参与团队技术分享。

5. 协作支持

与开发团队沟通，协助验证修复方案，确保合约安全性。

6. 文档整理

维护审计相关文档，记录分析过程和结果。

岗位要求

1. 基础知识

  • 熟悉区块链技术和以太坊生态，了解智能合约开发原理。
  • 熟悉 Solidity 编程语言，了解常见智能合约设计模式。

2. 技术能力

  • 了解智能合约常见安全漏洞，如重入、整数溢出、未授权访问等。
  • 有使用智能合约审计工具经验者优先，如 Mythril、Slither、Truffle。
  • 熟悉 EVM（以太坊虚拟机）运行机制者优先。

3. 编程与工具

  • 熟悉至少一种编程语言，如 Python、JavaScript，用于脚本编写或工具开发。
  • 熟悉 Git 版本控制，了解 Linux 基本操作。

4. 学习能力

  • 具备较强的学习能力和问题解决能力，对区块链安全有浓厚兴趣。
  • 能够阅读英文技术文档，跟踪国际社区最新动态。

5. 其他要求

  • 计算机、密码学、区块链或相关专业在读学生，本科及以上。
  • 每周可实习 3-5 天，实习期 3 个月以上。
  • 良好的沟通能力和团队合作意识。

加分项

  • 有区块链项目开发或审计经验。
  • 参与过 CTF（Capture The Flag）或安全相关比赛。
  • 熟悉 DeFi 协议或 NFT 项目的工作原理。
  • 了解其他区块链平台经验者优先，如 Solana、Binance Smart Chain。
