web3学习路径

# 阶段 0：Web3 入门基础（1–2 周） ## ▸ 必学内容 ### ● 创建钱包 & Web3 基础 + MetaMask 入门 - [https://support.metamask.io/zh-cn/start/getting-started-with-metamas

来源路径：web3/web3学习路径.md

阶段 0：Web3 入门基础（1–2 周）

▸ 必学内容

● 创建钱包 & Web3 基础

  • MetaMask 入门
    • https://support.metamask.io/zh-cn/start/getting-started-with-metamask/
    • https://learn.metamask.io/zh-CN/lessons/what-is-web3

● 智能合约基础概念

  • Solidity 官方入门
    • https://docs.soliditylang.org/zh-cn/latest/introduction-to-smart-contracts.html

● 工具环境

  • Remix 在线 IDE
    • https://remix.ethereum.org
  • 以太坊区块浏览器 Etherscan
    • https://etherscan.io

● 推荐课程

  • 北大区块链课程
    • https://www.bilibili.com/video/BV1Vt411X7JF

▸ Checkpoints

  • 创建钱包
  • 交易一次（swap）
  • 在 Etherscan 分析一笔交易
  • 用 Remix 部署 HelloWorld

阶段 1：Solidity + Foundry 核心基础（3–6 周）

▸ Solidity 核心语法

  • 官方文档
    • https://docs.soliditylang.org
  • ABI 编码
    • https://docs.soliditylang.org/en/latest/abi-spec.html
  • storage / memory / calldata
  • tx.origin / msg.sender
  • event / mapping / struct
  • fallback / receive

▸ EVM 基础（slot / SLOAD / SSTORE / ABI）

  • 参考
    • https://docs.soliditylang.org/en/latest/internals/layout_in_storage.html

▸ 开发工具链（主力）

● Foundry

  • 文档
    • https://book.getfoundry.sh

● 代理模式

  • Transparent Proxy
    • https://docs.openzeppelin.com/contracts/4.x/api/proxy#TransparentUpgradeableProxy
  • UUPS Proxy
    • https://docs.openzeppelin.com/contracts/4.x/api/proxy#UUPSUpgradeable
  • Diamond Proxy (ERC2535)
    • https://eips.ethereum.org/EIPS/eip-2535

▸ Checkpoints

  • 手写 ERC20
  • 手写 Bank（存款/取款）
  • Foundry 单测 ≥10
  • Foundry fuzz 测试
  • 手写 3 类 Proxy
  • 部署到 testnet（Forge script）

阶段 2:DeFi 协议原理（2–4 周）

▸ 必学协议

● AMM / Uniswap

  • V2 Whitepaper
    • https://uniswap.org/whitepaper
  • V3 Whitepaper
    • https://uniswap.org/whitepaper-v3

● Curve

  • Docs
    • https://curve.readthedocs.io

● GMX

  • Docs
    • https://gmxio.gitbook.io

● 推荐课程

  • 梁培利 DeFi 课程

▸ Checkpoints

  • 理解 x*y=k
  • 用 Solidity 写一个最简 AMM
  • 理解 LP、滑点、套利
  • 用 Foundry 模拟一次套利

阶段 3：Solidity 漏洞体系（2–3 周）

▸ 必学漏洞类型

  • 重入攻击（Reentrancy）
    • https://blog.openzeppelin.com/reentrancy-after-istanbul
  • 授权错误（AccessControl）
    • https://docs.openzeppelin.com/contracts/4.x/access-control
  • 调用陷阱（delegatecall）
    • https://docs.soliditylang.org/en/latest/introduction-to-smart-contracts.html#delegatecall
  • 预言机操控
    • https://chain.link/education
  • TWAP 操控
    • https://samczsun.com/twap-oracle-attacks
  • 闪电贷攻击（Flashloan）
    • Aave Docs: https://docs.aave.com/faq/flash-loans

▸ Checkpoints

  • 每种漏洞写出简版复现
  • 用 Foundry 写 PoC

阶段 4：靶场训练（4–8 周）

▸ 你必须刷完的 3 套靶场

● Ethernaut

  • https://ethernaut.openzeppelin.com

● Damn Vulnerable DeFi

  • https://www.damnvulnerabledefi.xyz/

● Paradigm CTF（选做）

  • https://github.com/paradigmxyz/paradigm-ctf-2021

▸ Checkpoints

  • Ethernaut ≥ 20 关
  • DVDeFi 全部完成
  • Paradigm 完成几题
  • 写 3 个漏洞 PoC

阶段 5：真实攻击事件复现（4–6 周）

▸ 工具（必须掌握）

  • 区块链浏览器（Etherscan）
    • https://etherscan.io
  • Phalcon Explorer（调用栈、资金流）
    • https://explorer.phalcon.xyz
  • Foundry mainnet fork
    • https://book.getfoundry.sh/forge/forking

▸ 推荐复现仓库

  • DeFiHackLabs
    • https://github.com/SunWeb3Sec/DeFiHackLabs
  • learn-evm-hacks
    • https://github.com/1nchboy/learn-evm-hacks

▸ Checkpoints

  • 复现 ≥ 3 个真实攻击事件
  • 写 ≥ 3 个攻击 PoC
  • 在 Phalcon 分析 ≥5 笔攻击

阶段 6：审计报告学习（2–3 周）

▸ 报告来源（按优先级）

  • Solodit（合集）
    • https://solodit.xyz
  • Code4rena
    • https://code4rena.com
  • Sherlock
    • https://audits.sherlock.xyz
  • Cantina
    • https://cantina.xyz
  • Trail of Bits 博客
    • https://blog.trailofbits.com

▸ Checkpoints

  • 阅读 ≥15 份报告
  • 对 ≥5 个漏洞写复盘
  • 仿写 2 份完整审计报告

阶段 7：Move / Sui / Aptos（4–8 周）

▸ 学习资源

  • Move Book
    • https://move-book.com
  • Move Audit Resources
    • https://github.com/0xriazaka/Move-Audit-Resources
  • Aptos 官方教程
    • https://learn.aptoslabs.com/en
  • Sui / Move 安全（Spearbit）
    • https://x.com/spearbit/status/1900610959812776221
  • Move Prover
    • https://medium.com/@monethic/getting-started-with-move-prover-for-aptos-01923d5a1d0f
  • OtterSec Move 审计报告
    • https://ottersec.notion.site/Sampled-Public-Audit-Reports-a296e98838aa4fdb8f3b192663400772

▸ Checkpoints

  • 写一个 Move coin
  • 写一个 Move AMM
  • 阅读 ≥3 个 Move 审计报告
  • Move Prover 验证 invariant

阶段 8：Bounty / 审计比赛（持续）

▸ 赏金平台

  • Immunefi
    • https://immunefi.com
  • HackenProof
    • https://hackenproof.com

▸ 审计比赛

  • Code4rena
    • https://code4rena.com
  • Sherlock
    • https://audits.sherlock.xyz

▸ Checkpoints

  • 提交 ≥1 个有效漏洞
  • 参加 ≥1 次审计比赛
  • 写 ≥1 份正式审计报告

最终你将具备的能力

  • Solidity + Foundry + 代理模式
  • DeFi 深入理解
  • 漏洞体系知识
  • 靶场攻克能力
  • 主网攻击复现能力
  • 能读写审计报告
  • Move / Aptos / Sui 审计基础
  • 能参与 Code4rena / Sherlock / Immunefi 挖洞
